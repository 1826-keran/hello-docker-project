
# Hello Captain Docker Project

This is my first Docker project for roadmap.sh.

## Requirements
* Base image :'alpine:latest'
* Action: Prints "Hello, Captain!" before exiting
  
## How to run
1.Build the image:
 'docker build -t hello-captain .'
2.Run the container:
'docker run hello-captain'
